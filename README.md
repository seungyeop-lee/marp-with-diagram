# marp-with-diagram

[marp](https://marp.app/)를 이용한 PDF 변환 시, Diagram 삽입을 추가

- 사용가능한 Diagram은 https://kroki.io/#support 참고

## 필요 환경

- docker
- bash
- node
- 비어있는 18000번 포트

## 사용법

```
npm install
```

```
./run.sh [markdown file path]
```
